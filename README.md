# Lex/bison parser
Para correr el codigo, necesitas tener instalados los siguientes programas:
> Compilador GCC \n
> Flex Language \n
> Bison Language \n
> Valgrind \n

Utiliza el comando:
```
   make
```
Para generar el ejecutable

Para probar fugas de memorias con valgrind, utiliza:
```
   make valgrind
```
